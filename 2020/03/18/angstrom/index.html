Reasonably Strong Algorithm

RSA strikes again! This is a basic RSA problem that provides the following values.

A quick reminder on RSA

Value Description
p is a prime number
q is a prime number
n p * q
private key ((p - 1) * (q - 1 )) invert e
e public key
totient(n) = phi (p-1) * (q-1)
ciphertext = (c) pow(m, e, n)
plain text = (m) pow(c, e, n)
